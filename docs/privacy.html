Política de Privacidade — PolySpeaker

Última atualização: 1 de julho de 2026

O PolySpeaker é um aplicativo de aprendizado de idiomas desenvolvido por Raul Lima. Esta política descreve como tratamos as informações dos usuários.

Dados coletados

O PolySpeaker não coleta, armazena nem compartilha dados pessoais dos usuários. Todo o progresso de aprendizado é salvo localmente no seu dispositivo e nunca enviado a servidores externos.

Funcionalidade de IA (chat de dúvidas)

O recurso "Tirar dúvida com IA" envia a frase em questão para a API da DeepSeek exclusivamente para gerar uma resposta gramatical. Nenhuma informação de identificação do usuário é transmitida. As perguntas não são armazenadas pelo PolySpeaker.

Microfone

O app solicita acesso ao microfone somente para o recurso de reconhecimento de pronúncia. O áudio é processado localmente pelo sistema operacional e não é gravado nem enviado a terceiros pelo PolySpeaker.

Serviços de terceiros

O app utiliza Firebase Cloud Messaging exclusivamente para envio de notificações locais de lembrete. Nenhum dado pessoal é associado a essas notificações.

Crianças

O PolySpeaker não coleta intencionalmente dados de crianças menores de 13 anos.
